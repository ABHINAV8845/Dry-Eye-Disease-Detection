# 👁️ Dry Eye Disease Detection using AI
AI-based web application that predicts Dry Eye Disease using gene expression data.

This project predicts Dry Eye Disease using gene expression data using Machine Learning.

---

-->🚀 Features
- Upload gene expression CSV file
- Predict Dry Eye / No Dry Eye
- Show confidence score
- Provide basic health advice


-->🧠 Technologies Used
- Python
- Scikit-learn
- Streamlit
- Pandas
- NumPy


-->⚙️ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Create models:
   python create_dummy_models.py

3. Run app:
   streamlit run app.py

---

--> 📊 Sample Files
- dry_eye_sample.csv
- sample_gene_expression.csv
